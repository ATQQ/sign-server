# sign-server

活动考勤助手小程序-服务端

![图片](https://img.cdn.sugarat.top/mdImg/MTYxNzYwODI3MDQ4OA==617608270488)


仓库基于[express-template](https://github.com/ATQQ/express-template)进行二次开发
## 使用

1. 安装依赖
```sh
yarn install
```

2. 开发环境运行
```
yarn dev
```

3. 生产环境运行
```
yarn start
```

4. 格式化代码
```
yarn lint
```

5. 跑测试
```
yarn test
```
